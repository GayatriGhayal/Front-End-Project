CSS Box Model

The CSS Box Model is a fundamental concept for understanding how elements are displayed and interact with each other on a webpage. It consists of four areas: content, padding, border, and margin.

Box Model Breakdown

  • Content: The actual content of the box, such as text or images.
  • Padding: Clears an area around the content, inside of the border.
  • Border: A border surrounding the padding (if specified).
  • Margin: Clears an area outside the border, creating space between elements.

Example


/* CSS Box Model */
div {
  width: 300px;
  padding: 20px;
  border: 5px solid black;
  margin: 10px;
}
