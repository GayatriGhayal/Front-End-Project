CSS Colors

CSS allows you to apply colors to your HTML elements to create visually appealing websites. Colors can be defined in different formats like names, HEX, RGB, and HSL.

Color Formats

  • Color Names: Example: red, green, blue.
  • HEX Codes: Example: #ff0000 (red), #00ff00 (green).
  • RGB: Example: rgb(255, 0, 0) for red.
  • RGBA: Example: rgba(255, 0, 0, 0.5) for semi-transparent red.
  • HSL: Example: hsl(0, 100%, 50%) for red.

Example


/* Using color names */
body {
  background-color: lightblue;
}

/* Using HEX code */
h1 {
  color: #ff0000;
}

/* Using RGB */
p {
  color: rgb(0, 128, 0);
}
