CSS Flexbox

CSS Flexbox is a powerful layout model that provides a simpler way to align elements and distribute space within a container. It's especially useful for creating responsive web designs.

Flexbox Properties

  • display: flex; Defines a flex container.
  • justify-content: Aligns items horizontally (e.g., flex-start, center, space-between).
  • align-items: Aligns items vertically (e.g., flex-start, center, stretch).
  • flex-wrap: Allows items to wrap onto a new line when necessary.

Example (Flexbox Layout)


/* Flexbox Layout Example */
.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
}

.item {
  width: 30%;
  background-color: lightgreen;
  padding: 20px;
}
