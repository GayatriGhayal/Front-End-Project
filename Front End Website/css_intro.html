Introduction to CSS

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to change the color, font, spacing, and position of elements on a webpage.

Why CSS?

  • CSS helps separate content from design, improving website maintainability.
  • It allows developers to create visually appealing and responsive designs.
  • CSS enhances accessibility by enabling changes to font sizes and layout for different devices.

How CSS Works

CSS is applied to HTML elements by linking an external stylesheet, using the <style> tag, or applying inline styles directly within the element.

Example


<style>
  body {
    background-color: lightblue;
    color: darkblue;
  }
</style>
