CSS Layout

CSS Layout is used to arrange elements in a page. There are several techniques for layout in CSS including Flexbox, Grid, and positioning.

Common Layout Techniques

  • Flexbox: A layout model that allows you to design complex layouts with a simple, flexible box container.
  • CSS Grid: A 2D grid-based layout system for creating web page structures.
  • Float and Positioning: Traditional methods of creating layouts using the `float` property and `positioning` properties (relative, absolute, fixed).

Example (Flexbox Layout)


/* Flexbox Layout Example */
.container {
  display: flex;
  justify-content: space-between;
}

.item {
  width: 30%;
  background-color: lightblue;
  padding: 20px;
}
