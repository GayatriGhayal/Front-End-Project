CSS Selectors

CSS selectors are patterns used to select the HTML elements you want to style. They can be simple or more complex, targeting specific elements based on their type, class, ID, or other attributes.

Common Selectors

  • Type Selector: Selects elements based on the element type. Example: <div>, <p>.
  • Class Selector: Selects elements with a specific class attribute. Example: .class-name.
  • ID Selector: Selects an element with a specific ID attribute. Example: #id-name.
  • Universal Selector: Selects all elements. Example: *.

Example


/* Type selector */
p {
  font-size: 16px;
}

/* Class selector */
.button {
  background-color: blue;
  color: white;
}

/* ID selector */
#header {
  text-align: center;
}
