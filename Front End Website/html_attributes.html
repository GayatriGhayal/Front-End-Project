HTML Attributes

HTML attributes provide additional information about HTML elements. They are added inside the opening tag of an element.

Common HTML Attributes

  • href: Specifies the URL for a link in the <a> tag.
  • src: Specifies the path to an image in the <img> tag.
  • alt: Provides alternative text for images.
  • id: Specifies a unique identifier for an element.
  • class: Assigns a class name to an element for styling.

Example


<a href="https://example.com">Visit Example</a>
<img src="image.jpg" alt="Sample Image">
