HTML Elements

HTML elements are the building blocks of web pages. An element typically consists of a start tag, content, and an end tag.

Common HTML Elements

  • <h1> to <h6>: Header tags used for different levels of headings.
  • <p>: Paragraph element used for text blocks.
  • <img>: Embeds an image on a webpage.
  • <a>: Anchor element used to create hyperlinks.
  • <div>: Defines a division or section within a document.
  • <span>: Inline container used for grouping elements.
