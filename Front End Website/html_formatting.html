HTML Formatting

HTML offers various tags to format content, such as for changing text styles or adding space between elements.

Formatting Tags

  • <b>: Makes text bold.
  • <i>: Italicizes text.
  • <u>: Underlines text.
  • <em>: Emphasizes text (usually italicized).
  • <strong>: Specifies strong importance (usually bold).

Example


<p><strong>This is bold text</strong> and this is <em>italic text</em></p>
