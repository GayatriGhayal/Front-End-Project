Introduction to HTML

HTML stands for HyperText Markup Language. It is used to create the structure of web pages. HTML consists of various elements, tags, and attributes that help in organizing content on the web.

Basic HTML Structure

Every HTML document starts with a doctype declaration and a root element ``. The structure includes two main parts: `` and ``. Here's an example of basic HTML structure:


<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8">
    <title>My First Webpage</title>
  </head>
  <body>
    <h1>Welcome to HTML!</h1>
    <p>This is an introductory page for HTML.</p>
  </body>
</html>

Why Learn HTML?

HTML is the foundation of web development. It is used to structure all content on a webpage, such as text, images, links, and multimedia. Understanding HTML is crucial to mastering web development and design.
