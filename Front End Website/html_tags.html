HTML Tags

HTML tags define the structure of a webpage. These tags are enclosed in angle brackets (<>) and typically come in pairs: an opening tag and a closing tag.

Common HTML Tags

  • <h1> to <h6>: Headings that define the structure of content from most important (<h1>) to least important (<h6>).
  • <p>: Paragraph tag to create blocks of text.
  • <a>: Anchor tag used to define hyperlinks.
  • <img>: Image tag used to embed images on the page.
  • <div>: Generic container element used for grouping and styling sections of the page.
